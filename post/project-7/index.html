Note:

  1. Scikit learn requires numeric data and no missing values.
  2. Real world data often has missing values so we will preprocess data often eg you recieve a color as your data, this makes it categorica and as scikit learn will not accept categorical, you’ll need to convert it to numerical.

You can create binary features from the categories, this is abinary preprocession called Dummy Variables.

To create dummy variables, we can use scikit-learn: OneHotEncoder or pandas: get_dummies()
